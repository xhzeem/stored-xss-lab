Getting Started

Welcome to the HR Portal! This internal system allows you to manage various HR-related tasks.

Features

  • Applicants: Track job applications and candidate information
  • Interviews: Record and review interview notes
  • Employee Bios: Browse employee profiles
  • Job Postings: View and post open positions
  • Announcements: Company-wide news and updates
  • Calendar: Upcoming events and meetings
  • HR Tickets: Submit support requests
  • Audit Log: System activity tracking
  • Onboarding: New employee task management
  • Wiki: Internal documentation
