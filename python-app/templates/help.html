Getting Started

ProjectHub is a project management suite designed for teams. Use the navigation bar to access different features.

Features

  • Projects - Create and manage project workspaces
  • Notes - Team notes for collaboration and documentation
  • Kanban - Visual task board with drag-and-drop columns
  • Wiki - Project documentation and knowledge base
  • Reports - Generate and review project reports
  • Contacts - Contact directory for stakeholders
  • Calendar - Schedule events and deadlines
  • Settings - Configure project preferences

API Endpoints

  • GET /health - Health check endpoint
  • GET / - Dashboard
  • GET|POST /projects - List/create projects
  • GET /project/<id> - View project
  • GET|POST /notes - List/create notes
  • GET|POST /kanban - Kanban board
  • GET|POST /wiki - Wiki pages
  • GET|POST /reports - Reports
  • GET|POST /contacts - Contacts
  • GET|POST /calendar - Calendar
  • GET|POST /settings - Settings
